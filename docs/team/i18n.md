# Internationalisation (FR / EN)

Le site est **français par défaut**, avec un **toggle FR|EN en haut à droite**
(topbar). Le choix est mémorisé dans `localStorage` (`tm_lang`) et réappliqué à
chaque visite. Pas d'auto-détection : le défaut reste le français.

## Architecture

- `src/ui/i18n.js` — cœur du système, sans dépendance :
  - `t('Texte français', { var })` : traduit une chaîne à la volée.
  - `applyTranslations(root)` : traduit le DOM **statique** par correspondance
    du texte français (aucune annotation HTML nécessaire).
  - `startAutoTranslate()` : un `MutationObserver` traduit automatiquement le
    contenu **inséré dynamiquement** (boutique, profil, mercato, dialogues…).
  - `setLang`, `getLang`, `onLangChange`, `mountLangToggle`.
- `src/ui/i18n-en.js` — dictionnaire **Français → Anglais** de l'application
  (import à effet de bord depuis `main.js`).
- `src/ui/i18n-legal-en.js` — dictionnaire des **pages légales** (terms,
  privacy, reset-password), importé par ces pages.

## Modèle « source française »

La **clé de traduction est le texte français** lui-même. Avantages : le code et
le HTML restent lisibles en français, et une entrée manquante se rabat
proprement sur le français (jamais de `undefined` à l'écran). Les paragraphes
multi-lignes sont tolérés (les espaces sont normalisés pour la recherche).

## Ajouter / traduire une chaîne

1. **Texte statique du HTML** : rien à baliser. Ajoute simplement la paire
   `'Texte français': 'English text'` dans `i18n-en.js` — l'observer et
   `applyTranslations` s'en chargent.
2. **Chaîne construite en JS** : enveloppe-la dans `t('…')`, avec des
   variables si besoin : `t('Utiliser : {power}', { power })`. Puis ajoute la
   clé dans `i18n-en.js`.
3. **Page légale** : ajoute la paire dans `i18n-legal-en.js` (clé = texte
   français, les espaces multiples sont normalisés automatiquement).

## Après modification

`node build.js` échoue dans l'environnement sandbox (suppression interdite) —
synchroniser `src/` → `public/src/` par recopie en place :
`cp -rf src/. public/src/`. Committer ensuite `public/src/`.
